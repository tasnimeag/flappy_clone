Ce projet est un jeu simple, inspiré de **Flappy Bird**, développé avec le framework **Phaser.js**. Le joueur contrôle un oiseau et doit appuyer sur la **barre d'espace** pour le faire sauter et éviter des obstacles (tuyaux) générés aléatoirement. Le jeu intègre des mécaniques essentielles telles que la gravité, la détection des collisions, le suivi du score, la génération procédurale d'obstacles et la possibilité de recommencer. Ce projet illustre les fondamentaux du développement de jeux 2D en JavaScript avec Phaser, notamment la gestion des sprites, la physique, les animations et la gestion des événements.
